# Rice Thoughts Blog 🍚

*A guy who likes rice and thinks.*

Rice Thoughts is a content-focused blog platform for publishing essays on politics, culture, and current events. Built with Next.js, React, and TypeScript, the application uses Wisp CMS for content management and dynamic post rendering.

## Features

* Dynamic blog post rendering with Wisp CMS
* Responsive and modern UI
* Modular page architecture for scalable development
* Structured content management
* SEO-friendly Next.js application

## Tech Stack

* Next.js
* React
* TypeScript
* Wisp CMS
* Tailwind CSS

## Future Improvements

* Tag-based filtering and content discovery
* Search functionality
* Related posts recommendations
* Enhanced content categorization

## Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000` to view the application locally.

## Project Goals

This project was built to explore content-driven web applications, CMS integration, and scalable frontend architecture while providing a platform for long-form writing and essays.
